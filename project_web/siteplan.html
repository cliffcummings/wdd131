The new web pages are intended to use proper techniques learned in class to both simplify and improve the web pages located at www.sunburst-design.com

Step 1 - Setup

This file: siteplan.html

Step 2 - Site Content

  1. Site Name: The Sunburst Design Home Page
  2. Site Purpose: Descirbe Sunburst Design & Paradigm Works services and training. Offer free downloadable copies of Cliff's conference papers. These conference papers have been freely downloadable for 30+ years. By word of mouth, engineers have found, accessed, and read Cliff's papers. The engineers also discover that Sunburst Design offers training on these topics. This has worked as a technical marketing tool to help gain training business.
  3. Scenarios:
    • How can I download Cliff's papers? (At the end of each paper is a short description of the types of training offered by Sunburst Design)
    • What types of training are offered by Sunburst Design?
    • How can I contact Suburst Design to request information about trianing?
  4. Color Schema: The colors are defined in the :root element at the top of styles/sunburst.css file. I am currently using --sunburst_red: for the logo text and --back_color1: for the footer. More colors will be used as the project progresses
  5. Typography:
    I explicityl use Roboto family in both the body and h1-header
